Key Features
✓Flashcard content: Each side has text (required) + optional image. Images stored in Supabase Storage.
✓Streak scoring: Correct answer → streak +1. Wrong → streak resets to 0. Streak ≥ 10 → marked memorized.
✓Weighted selection: Weight = 1/(streak+1). Low streak questions are prioritized in session sampling.
✓Memorized exclusion: Memorized questions excluded by default. Toggle to include when configuring a session.
✓Session re-queue: Wrong answers are re-added to end of queue. Session ends only when all questions are answered correctly.
✓Keyboard shortcuts: Space → reveal answer. ← Missed it. → Got it.
✓Dark/light mode: System-aware via Mantine ColorScheme.
✓Auth: Google OAuth + email/password via Supabase Auth. Server-side route protection via Next.js middleware.
